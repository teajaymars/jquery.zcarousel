jquery.zcarousel

simple image carousel

Download

  • jquery.zcarousel.min.zip  — production (7kb)
  • jquery.zcarousel.zip  — development (9kb)

Demonstration

Features

  • Streaming - Images are downloaded when they are displayed, saving the user from a megabytes of data when they visit your page.
  • Compatible - Tested on a range of browsers; certified compatible with Internet Explorer 7, 8, and 9!
  • Built with jQuery - turn any element into a carousel with one function call.
  • Lightweight - the code, stylesheets and dependancies are well below 10kb.
  • Extensible - zCarousel does One Thing Well™. Want to change the animation? The code is small, easy to read, and simple to modify.
  • Openly Licensed - Released under the MIT license. You are free to use it as you please :-)

Usage

$('#carousel-div').zcarousel(
[
  { caption: 'Here is the first image.',         
    url: 'http://somewhere.com/image1.jpg', 
    offset: '10px' 
  },
  { caption: 'Second image now appearing here.', 
    url: 'http://somewhere.com/image2.jpg', 
    offset: '0'    
  }
] );
